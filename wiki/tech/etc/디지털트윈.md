# 디지털 트윈 (Digital Twin)

## 1. 개요 (Overview)
- **기술명:** 디지털 트윈 (Digital Twin)
- **정의:** 물리적 자산·프로세스·시스템을 실시간으로 반영하는 가상 디지털 복제 모델로, 시뮬레이션·예측·최적화를 가능하게 하는 기술
- **배경:** NASA가 2010년대 초 우주선 유지보수에 개념을 도입했고, IoT·AI·클라우드 기술의 발전으로 제조·건설·스마트시티 등 산업 전반으로 확장됐다.

## 2. 주요 특징 (Key Features)
- **실시간 동기화:** IoT 센서 데이터로 물리 자산 상태를 디지털 모델에 실시간 반영
- **시뮬레이션:** 실제 운영 전 시나리오 테스트 (부하 증가, 장애 예측 등)
- **양방향 연동:** 디지털 모델에서 내린 결정을 물리 시스템에 다시 적용 가능
- **[[디지털쓰레드]]와 연계:** 제품 수명주기 전반의 데이터 흐름을 디지털 트윈이 집약

## 3. 작동 원리 (How it Works)
1. IoT 센서·카메라·PLC 등에서 물리 자산의 실시간 데이터 수집
2. 디지털 모델(3D CAD, 물리 시뮬레이션, AI 모델)에 데이터 반영
3. 이상 감지·예측 정비·최적화 결과 생성
4. 결과를 물리 시스템의 제어 시스템에 피드백

## 4. 주요 활용 사례 (Use Cases)
- 사례 1: 스마트 제조 — 생산 라인 디지털 트윈으로 불량률·가동률 최적화
- 사례 2: 스마트 빌딩 — 에너지 소비·공조 시스템 실시간 최적화
- 사례 3: 항공기 엔진 — GE의 Predix 플랫폼 기반 엔진 상태 실시간 모니터링·예측정비

## 5. 장단점 (Pros & Cons)
### 장점 (Pros)
- 실제 자산 변경 없이 시뮬레이션으로 리스크 사전 제거
- 예측 정비로 다운타임 및 유지보수 비용 절감
### 단점 (Cons)
- 정밀한 디지털 트윈 구축에 높은 초기 비용
- IoT 데이터 품질·레이턴시에 따라 모델 정확도 좌우
