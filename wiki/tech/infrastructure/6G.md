# 6G

## 1. 개요 (Overview)
- **기술명:** 6G (6th Generation Mobile Communication)
- **정의:** 2030년대 상용화 목표의 차세대 통신 기술로, 단순 속도 향상을 넘어 AI를 설계 단계부터 내재화(AI-Native)하고 의미 기반 통신(시맨틱 커뮤니케이션)을 구현하는 '지능형 디지털 신경망'
- **배경:** 5G가 물리적 연결의 한계(품질 불균형, Shannon 한계 접근)에 봉착하면서, 데이터를 단순히 옮기는 파이프가 아닌 의미와 맥락을 전달하는 인지 네트워크로의 진화가 필요해졌다.

## 2. 핵심 혁신 키워드

### AI-Native (네이티브 AI)
- 기존: AI가 네트워크 운영 보조 도구 (AI for Operations)
- 6G: 설계 단계부터 AI가 내재화 → 네트워크 자체가 분산형 컴퓨터로 기능
- **AI-RAN:** 초대규모 MIMO, 테라헤르츠 대역의 복잡한 채널 관리에 AI 필수
- 딥러닝 기반 CSI(채널 상태 정보) 피드백 압축 → 상향링크 자원 절감

### 시맨틱 커뮤니케이션 (Semantic Communication)
- 데이터 비트(bit) 대신 **의미(meaning)** 전송
- 텍스트 전송 시 기존 대비 100배 이상 압축 효율
- Shannon 한계(섀넌 한계)를 극복하는 유일한 대안으로 평가

### 에이전틱 AI와 자율 네트워크
- **Level 5 완전 자율 네트워크**: 운영자 개입 없이 AI가 계획·실행·복구
- 목표 지향적 자율성: "VIP 고객 품질 최우선" 같은 고수준 명령만 주면 AI가 처리
- 자가 치유: 장애 발생 시 수 초 내 논리 추론으로 원인 분석 및 복구

## 3. 작동 원리 (How it Works)

**시맨틱 통신 기술:**
- **Deep JSCC (결합 소스-채널 코딩):** 딥러닝으로 데이터 압축과 오류 정정을 단일 신경망으로 통합 → 채널 상태 악화 시 절벽 효과(Cliff Effect) 방지, 완만한 성능 저하(Graceful Degradation) 보장
- **공유 지식 베이스:** 송·수신 측이 사전 학습된 모델·지식 그래프를 공유 → 완전한 데이터 대신 의미 정보만 전송

**자율 네트워크 계층:**
기지국·코어망의 AI 에이전트들이 실시간 정보 교환 → 트래픽 과부하 해소, 장애 자가 복구

## 4. 주요 활용 사례 (Use Cases)
- **모빌리티:** 센서 원본 대신 "전방 사고 발생" 같은 의미 정보만 교환 → 초저지연 자율주행
- **디지털 헬스:** 영상 원본 대신 "낙상 감지" 추론 결과만 전송 → 데이터 효율 + 프라이버시 보호
- **초실감 미디어:** 홀로그램 시맨틱 정보만 전송, 수신단 생성형 AI가 배경 복원 → 끊김 없는 몰입형 경험
- **스마트 팩토리:** 산업 현장 AI 에이전트들의 협력적 의사결정 네트워크

## 5. 장단점 (Pros & Cons)
### 장점 (Pros)
- 대역폭·전력 효율 획기적 향상 (100배 압축 등)
- 네트워크 운영 자동화로 관리 비용 절감
- AI 추론을 네트워크 인프라에 분산 → 엣지 AI 가능
### 단점 (Cons)
- **이종 기기 간 지식 베이스 동기화** 표준화 문제
- AI 환각(Hallucination)이 네트워크 제어 로직에서 발생 시 **네트워크 마비(Network Paralysis)** 위험
- AI 운영의 막대한 전력 소모 → **그린 6G** 기술 필요
- AI 판단 근거의 불투명성 → **설명 가능한 AI(XAI)** 요구

## 6. 시장 전망
- **AI-RAN 시장:** 2037년까지 연평균 **29.4% 성장**, 154억 달러 규모 전망
- **3GPP Release 20:** 2029년경 6G 기술 규격 윤곽 발표 예정
- 주요 경쟁: 노키아(AI 네트워크 패브릭), 에릭슨(생성형 AI 운영), 화웨이(ADN 자율 주행 네트워크)
